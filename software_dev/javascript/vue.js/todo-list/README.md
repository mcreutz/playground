# Vue.js Todo List

A simple todo list app built with Vue 3 and TypeScript.

## Installation

Navigate to the project directory:
```bash
cd todo-list
```

Install dependencies:
```bash
npm install
```

## Run

```bash
npm run dev
```

Open http://localhost:5173 in your browser.

## Build

Creates production-ready files for deployment:
```bash
npm run build
```
